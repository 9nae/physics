# คู่มือครู: สื่อการสอน Physics เรื่องเสียง

คู่มือประกอบเว็บบทเรียนหัวข้อ 12.3 ปรากฏการณ์เกี่ยวกับเสียง สำหรับใช้สอนหรือให้นักเรียนเรียนด้วยตนเอง

## ภาพรวมบทเรียน

สื่อเว็บนี้แบ่งเป็น 4 หน่วยหลัก มี `Guided Lesson Player` สำหรับให้นักเรียนเรียนทีละฉาก พร้อมเสียงบรรยาย TTS, เสียงทดลอง, quiz เฉลยทันที และมีส่วนอ่านทบทวนแบบเอกสาร

1. คลื่นนิ่งของเสียง
2. การสั่นพ้องของอากาศในท่อปลายปิดหนึ่งด้าน
3. บีต
4. ปรากฏการณ์ดอปเพลอร์และคลื่นกระแทก

## เป้าหมายการเรียนรู้

นักเรียนควรทำได้ดังนี้

1. อธิบายว่าคลื่นนิ่งของเสียงเกิดจากการซ้อนทับของเสียงตกกระทบและเสียงสะท้อน
2. ระบุตำแหน่งบัพและปฏิบัพ ทั้งในมุมของความดันเสียงและการกระจัดของโมเลกุลอากาศ
3. คำนวณความยาวคลื่น ความถี่ และอัตราเร็วเสียงจากความสัมพันธ์ `v = fλ`
4. อธิบายการสั่นพ้องของอากาศในท่อปลายปิดหนึ่งด้าน และใช้สมการ `f_n = nv/4L` เมื่อ `n = 1, 3, 5, ...`
5. อธิบายการเกิดบีตและคำนวณ `f_b = |f_1 - f_2|`
6. อธิบายดอปเพลอร์ของเสียงจากการเคลื่อนที่สัมพัทธ์ของแหล่งกำเนิดและผู้ฟัง
7. อธิบายคลื่นกระแทก มุมมัค และซอนิกบูมเมื่อแหล่งกำเนิดเร็วกว่าเสียง

## แผนการใช้ในชั้นเรียน

ก่อนเริ่ม ให้เปิด `index.html` แล้วกด `เริ่มเรียนทีละฉาก` นักเรียนสามารถกด `ฟังคำบรรยาย` เพื่อให้ browser อ่านบทพูดของครูเป็นภาษาไทยได้ ถ้าเครื่องไม่มีเสียงไทย ให้ติดตั้ง voice ภาษาไทยใน Windows หรืออ่านสคริปต์บนหน้าจอแทน

### คาบที่ 1: คลื่นนิ่งของเสียง

เริ่มจากทบทวนคลื่นนิ่งในเส้นเชือก แล้วเชื่อมมายังเสียงในอากาศ เปิด Simulator 1 และตั้งค่า เช่น ความถี่ 3000 Hz ระยะลำโพง-พื้น 60 cm ให้นักเรียนสังเกตว่าตำแหน่งดังและค่อยสลับกันอย่างไร

คำถามนำอภิปราย:

- เพราะเหตุใดพื้นแข็งจึงทำให้เกิดเสียงสะท้อนได้ดี
- ตำแหน่งเสียงดังที่สุดสัมพันธ์กับปฏิบัพความดันอย่างไร
- ถ้าระยะดัง-ค่อยถัดกันเพิ่มขึ้น แสดงว่าความยาวคลื่นเพิ่มขึ้นหรือลดลง

### คาบที่ 2: การสั่นพ้องในท่อ

ใช้ Simulator 2 ให้เปลี่ยนอุณหภูมิ ความถี่ และความยาวลำอากาศ โดยชี้ให้เห็นว่าเรโซแนนซ์ถัดกันต่างกัน `λ/2` และการสั่นพ้องครั้งแรกของท่อปลายปิดมี `L = λ/4`

ตัวอย่างโจทย์:

- อุณหภูมิ 28 °C ทำให้ `v = 331 + 0.6(28) = 347.8 m/s`
- ถ้าใช้เสียง 1000 Hz จะได้ `λ = 0.3478 m` และ `λ/2 ≈ 17.4 cm`
- ตำแหน่งเรโซแนนซ์ควรอยู่ใกล้ `λ/4, 3λ/4, 5λ/4, ...`

### คาบที่ 3: บีต

ใช้ Simulator 3 ตั้ง `f_1 = 392 Hz` และ `f_2 = 395 Hz` เพื่อให้ได้บีต 3 Hz จากนั้นลองเปลี่ยนเป็น 396, 400, 403 Hz ให้นักเรียนคำนวณผลต่างทีละคู่

จุดเน้น:

- บีตเปลี่ยนความดังตามเวลา
- การแทรกสอดของเสียงจากแหล่งอาพันธ์เปลี่ยนความดังตามตำแหน่ง
- ถ้า `f_b` มากเกินประมาณ 7 Hz หูมนุษย์มักแยกจังหวะบีตไม่ชัด

### คาบที่ 4: ดอปเพลอร์และคลื่นกระแทก

ใช้ Simulator 4 ตั้งแหล่งกำเนิดเคลื่อนที่เข้าหาผู้ฟัง แล้วให้สังเกตว่าหน้าคลื่นด้านหน้าถูกอัดและความถี่ที่ได้ยินสูงขึ้น ต่อด้วย Simulator 5 ให้เพิ่ม Mach number แล้วดูมุมมัคที่แคบลง

คำถามนำอภิปราย:

- ก่อนรถพยาบาลผ่าน ผู้ฟังได้ยินเสียงไซเรนสูงขึ้นหรือต่ำลง
- หลังรถพยาบาลผ่าน ความถี่เปลี่ยนไปทางใด
- ทำไมลมที่พัดคงตัว โดยที่แหล่งกำเนิดและผู้ฟังอยู่นิ่ง จึงไม่ใช่ดอปเพลอร์
- เมื่อ Mach number เพิ่มขึ้น มุมมัคเปลี่ยนอย่างไร

## เฉลยแบบตรวจสอบความเข้าใจ

1. คลื่นนิ่ง: ถ้าดัง-ค่อยถัดกันห่าง 10.00 cm จะได้ `λ/4 = 0.1000 m`, `λ = 0.4000 m`, `f = 350/0.4000 = 875 Hz`
2. สั่นพ้อง: ถ้าตำแหน่งเรโซแนนซ์ถัดกันห่าง 0.400 m จะได้ `λ/2 = 0.400 m`, `λ = 0.800 m`, `f = 346/0.800 = 433 Hz`
3. บีต: ถ้าเทียบกับ 392 Hz แล้วเกิดบีต 3 Hz ความถี่อีกตัวคือ 389 Hz หรือ 395 Hz
4. ดอปเพลอร์: ก่อนรถผ่านได้ยินความถี่สูงขึ้น ขณะผ่านใกล้ความถี่เดิม หลังรถผ่านได้ยินความถี่ต่ำลง
